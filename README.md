# Thrisha Kandi – Portfolio Website

This is my personal portfolio website built to showcase my skills, projects, and overall progress as a Computer Science student.

I wanted something simple, clean, and easy to navigate instead of an overly complex design.

## What’s inside

The website includes:

- A short introduction about me  
- My technical skills  
- Projects I’ve worked on  
- Education details  
- Internship / experience section  
- Contact information  

It’s fully responsive, so it works on mobile, tablet, and desktop.

## Tech stack

- HTML  
- CSS (Flexbox + Grid)  
- JavaScript  
- Font Awesome  
- Google Fonts  

## How to run

Just open `index.html` in your browser.  
No setup required.

## Customization

- Update content in `index.html`  
- Modify styles in `styles.css`  
- Add or remove sections as needed  

## Why I built this

To create a clean online portfolio and apply my frontend skills in a real project.
